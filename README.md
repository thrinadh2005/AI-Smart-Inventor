# 🛒 AI Smart Inventory Assistant

AI-powered inventory management for Indian shops with voice commands in English, Telugu, and Hindi.

## 🚀 **LOCAL LAUNCH - ONE CLICK**

### **For Windows:**
```cmd
powershell -Command "iwr -useb https://raw.githubusercontent.com/thrinadh2005/AI-Smart-Inventor/main/quick-start.bat | cmd"
```

### **For Linux/Mac:**
```bash
curl -sSL https://raw.githubusercontent.com/thrinadh2005/AI-Smart-Inventor/main/quick-start.sh | bash
```

### **Access Points:**
- **Desktop**: http://localhost:5173
- **Mobile**: http://YOUR_COMPUTER_IP:5173
- **API**: http://localhost:5000

## 🌟 Features

### 🎤 **Voice Commands**
- **English**: *"Add sale"*, *"Check stock of atta"*, *"Find cold drinks"*
- **Telugu**: *"అమ్మకం జోడించండి"*, *"ఆటా స్టాక్ తనిఖీ చేయండి"*
- **Hindi**: *"बिक्री जोड़ें"*, *"आटा का स्टॉक जांचें"*

### 🤖 **AI Intelligence**
- Smart demand prediction
- Seasonal adjustments
- Stockout warnings
- Reorder suggestions

### 📱 **Mobile Ready**
- Touch-friendly interface
- Works on any smartphone browser
- Voice input on mobile Chrome
- Offline capabilities

### 🇮🇳 **Indian Context**
- Local product names (atta, dal, oil)
- Indian Rupee (₹) support
- WhatsApp integration ready
- Multilingual interface
